# Digital Library Management System

This is a Python-based **Library Management System** developed using Object-Oriented Programming (OOP) concepts. It features a user-friendly GUI built with **Streamlit**.

## Features
* **Book Management:** Add new books to the library collection.
* **Search:** Easily search for books by title or author.
* **Borrow & Return:** Manage borrowing and returning processes with automatic availability updates.
* **View Collection:** View all available books and their current stock status.
